# Not-For-All (NSFW Providers)

A collection of NSFW and regional movie/series extensions for the Cosmix app.

## Included Providers

- **MlsbdProvider**: Fetches movies and series from MLSBD. Features automatic protected link un-wrapper (bypasses SaveLinks to find hidden GDFlix and HubCloud links) and exclusively fetches `.r2.dev` and HubCloud pure direct streams for instant, flawless playback!
- **ViralLinksProvider**: ViralLinksProvider
- **WowProvider**: Only NSFW Movies Available
- **WowUncutProvider**: WowUncut - Hindi, Bengali, Tamil & More Web Series

## Installation

You can add this repository to your Cosmix app to install the extensions. 
Open the Cosmix app, navigate to **Extensions > Add Repository**, and paste the following URL:

```text
https://raw.githubusercontent.com/cosmix-extensions/Not-For-All/builds/plugins.json
```
