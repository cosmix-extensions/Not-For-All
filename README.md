# MLSBD Provider

A Cosmix extension to fetch movies and series from MLSBD.

## Features
- Browsing main page
- Searching movies and series
- Automatic protected link un-wrapper (bypasses SaveLinks to find the hidden GDFlix and HubCloud links)
- **Direct Streaming:** Exclusively fetches `.r2.dev` and HubCloud pure direct streams for instant, flawless playback!

## Installation

You can add this repository to your Cosmix app to install the extension.
